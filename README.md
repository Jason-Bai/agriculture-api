# Agriculture API 项目说明

## 一、使用说明
* npm install
* npm start

## 二、接口说明

### 1. 用户信息接口
* POST    /api/users
* DELETE  /api/users/:userId
* GET     /api/users
* PATCH   /api/users/:userId

### 2. 登录,注册接口
* POST    /signin
* GET     /signup

### 3. 农业分类接口
* POST    /api/categories
* DELETE  /api/categories/:categoryId
* GET     /api/categories
* PATCH   /api/categories/:categoryId
